# 滑动窗口模板

```c++
int res = 0;
int l = 0;
int r = 0;
unordered_map<char, int> w;
while (r < s.size()) { // 右边界,一直扩展，直到窗口满足条件
    char c = s[r];
    w[c]++;
    r++;
    while (w[c] > 1) {// 左边界，维护一个满足条件的窗口
        char t = s[l];
        w[t]--;
        l++;
    }
    res = max(res, r - l); // 注意这儿是 r-l
}
```
